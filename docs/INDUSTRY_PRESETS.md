# Industry Presets — Pre-built Workflows per Vertical

Each preset configures: agent tone, outreach sequence, reply templates, email templates, voice scripts, doc checklist, compliance rules, dashboard layout.

## Construction / Contracting
- Tone: Assertive, direct
- Outreach: "I saw you're expanding — we fund construction companies up to $X"
- Triggers: hiring posts, new project announcements, equipment purchases
- Docs: bank statements, tax returns, contractor license
- Dashboard: pipeline-first view
